題型一：名詞解釋（10-14 分）

每年必考，5 題各 2 分。以下按出現頻率排列：

必考 Grignard reagent 110、109、107 Ch13 RX + Mg (in ether/THF) → RMgX。有機金屬化合物，碳端為親核性（Umpolung），可攻擊 C=O 生成醇。
必考 Base peak (Mass Spec) 110、109、107 Ch15 質譜中強度最高的峰（定義為 100%），其他峰以此為基準。不一定是 molecular ion (M⁺•)。

高頻 Alkoxymercuration-demercuration 110、109 Ch13 Alkene + Hg(OAc)₂ 在醇類溶劑中 → mercurinium ion → 醇（而非水）攻擊 → NaBH₄ 還原。產物為 ether（醚），Markovnikov 方向。
高頻 Crown ether (18-Crown-6 / 15-Crown-5) 110、109 Ch13 環狀聚醚，可螯合特定大小的金屬離子（18-Crown-6 螯合 K⁺、15-Crown-5 螯合 Na⁺）。使陰離子成為更好的親核劑（裸陰離子）。
高頻 Umpolung 109 Ch13 極性反轉。碳在 R-X 中是親電性的（δ+），但形成 Grignard (RMgX) 後變成親核性的（δ-），等同碳負離子 R⁻。
高頻 Spin-spin splitting (n+1 rule) 107 Ch16 ¹H NMR 中，鄰近碳上有 n 個等價 H，訊號被分裂成 n+1 個峰。原因：鄰近質子的核自旋產生局部磁場影響。
高頻 Protection/Deprotection of Alcohols 109 Ch13 保護：R-OH + TMSCl/base → R-O-TMS（SN2 at Si）。脫保護：R-O-TMS + TBAF(F⁻) 或 H₃O⁺ → R-OH。目的：防止 -OH 與 Grignard 反應。
高頻 McLafferty rearrangement 107 Ch15 含 C=O 的分子在 MS 中，γ-碳上的 H 經六員環過渡態轉移到 C=O 的 O 上，導致 β 斷裂。

Ch10 新增預測名詞（今年可能首次出現）

  • Lindlar's catalyst：被毒化的 Pd 催化劑（Pd/CaCO₃/Pb(OAc)₂），選擇性還原 alkyne → cis alkene (syn addition)
  • Dissolving metal reduction：Na 或 Li 溶在液態 NH₃ 中，還原 alkyne → trans alkene (anti addition)。經 radical anion 中間體。
  • Keto-enol tautomerism：酮-烯醇互變異構，enol 形式自動轉化為較穩定的 ketone。
  • Alkynide ion：Terminal alkyne 被強鹼（NaNH₂）去質子化 → R-C≡C⁻，是好的親核劑，可與 1° RX 進行 SN2。

題型二：IR + 分子式 → 推結構（4-6 分）

必考 C₆H₈ + IR absorption at 3300 cm⁻¹ → 推兩個可能結構

出現次數：110、109、107 三年都考（原題重複）

解題：

  • HDI = (12+2-8)/2 = 3（一個三鍵 = 2 + 一個其他）
  • 3300 cm⁻¹ = ≡C-H stretching → terminal alkyne
  • C₆H₈ 含 terminal alkyne 的可能結構：
    • 1-hexyne：CH≡C-CH₂-CH₂-CH₂-CH₃
    • 3,3-dimethyl-1-butyne：CH≡C-C(CH₃)₃
    • 或其他含 terminal ≡C-H 且 HDI=3 的 C₆H₈ 異構物

必考 C₄H₈O + IR 1720 cm⁻¹ + ¹H NMR triplet at δ 9.7 → 推兩個結構

出現次數：110、109、107 三年都考（原題重複）

解題：

  • HDI = (8+2-8)/2 = 1（一個雙鍵）
  • 1720 cm⁻¹ = C=O stretching
  • δ 9.7 = aldehyde H（-CHO 的 H 通常在 δ 9-10）
  • triplet → 這個 aldehyde H 旁邊有 2 個 H（n+1=3）
  • C₄H₈O 的 aldehyde：
    • Butanal：CH₃CH₂CH₂CHO → δ 9.7 的 H 被鄰近 CH₂ 的 2H 分裂成 triplet ✓
    • 2-Methylpropanal：(CH₃)₂CHCHO → δ 9.7 的 H 被鄰近 CH 的 1H 分裂成 doublet
  • 題目說 triplet，所以一個答案是 butanal（triplet），另一個是 2-methylpropanal（doublet）→ 列出兩個 aldehyde 異構物

題型三：MS + IR → 推未知碳氫化合物（6 分）

必考 給質譜 + IR 光譜 → 推結構並標示官能基

出現次數：110 Q4、109 Q4、107 Q7 → 每年都考

解題 SOP：

  1. 從 MS 找 M⁺•（分子離子峰）→ 分子量
  2. 算 HDI
  3. 看 IR 的 diagnostic region：
    • ~3300 sharp → ≡C-H（terminal alkyne）
    • ~2100-2200 → C≡C
    • >3000 → sp² C-H
    • <3000 → sp³ C-H only
  4. 用 MS 碎片驗證（差 15=CH₃、差 29=C₂H₅、差 14=CH₂）

題型四：預測反應產物 A~F（6 分）

必考 三組反應串聯，填 A~F

出現次數：110 Q5、109 Q5、107 Q5 → 三年幾乎同一題

(a) Ch13 醇的取代：

  • Cyclohexanol + TsCl / pyridine → A = cyclohexyl tosylate（OTs，好的離去基）
  • A + NaCN / DMSO → B = cyclohexyl cyanide（SN2，構型翻轉 inversion）

(b) Ch13 Grignard + 氧化：

  • Cyclopentanone + 1) CH₃MgBr 2) H₂O → C = 1-methylcyclopentanol（3° alcohol）
  • C + Na₂Cr₂O₇ / H₂O → D = No reaction（3° alcohol 不能被氧化！）

(c) Ch13 Grignard + ester + 氧化：

  • Lactone + 1) excess CH₃MgBr 2) H₃O⁺ → E = diol（兩個 -OH，其中兩個甲基來自 Grignard）
  • E + excess Na₂Cr₂O₇ / H₂O → F = 1° OH 被氧化成 carboxylic acid，2° OH 氧化成 ketone（若有的話）

題型六：pKa 酸性排序（6 分）

高頻 排列 4 個含 O-H 分子的 pKa

出現次數：109 Q7、107 Q7

用 ARIO 分析：

  • Carboxylic acid（pKa ~4-5）> Phenol（~10）> H₂O（15.7）> Alcohol（15-18）
  • 有 C=O 鄰近的 -OH（如 enol of β-diketone）→ 共振穩定 → 更酸
  • Induction：有 electron-withdrawing group（如 -NO₂、-Cl）→ pKa 降低

題型七：IR 區分 hexane / hexene / hexyne（4-6 分）

必考 三張 IR 光譜配對三種碳氫化合物

出現次數：110 Q8、109 Q8、107 Q8、107 小考 Q5 → 四份都考！

判斷要點：

化合物IR 特徵
Hexyne（terminal） ~3300 cm⁻¹ sharp（≡C-H stretch）+ ~2100 cm⁻¹（C≡C stretch）
Hexene >3000 cm⁻¹（=C-H stretch）+ ~1640 cm⁻¹（C=C stretch）+ 900-1000 cm⁻¹ bending
Hexane 全部 <3000 cm⁻¹（只有 sp³ C-H），1500-2000 區域無訊號

題型八：¹H + ¹³C NMR → 推結構（10 分大題）

必考 C₁₀H₁₄ + ¹H NMR 數據 → 推結構

出現次數：110 Q10a、109 Q10、107 Q10 → 三年都考一模一樣的題！

題目數據：

  • δ 7.0 (4H, broad singlet) → 對位取代苯環（para-disubstituted）
  • δ 2.85 (1H, septet, J=8 Hz) → CH 被 6 個等價 H 分裂（n+1=7）
  • δ 2.28 (3H, singlet) → CH₃ 直接接芳香環
  • δ 1.20 (6H, doublet, J=8 Hz) → 兩個等價 CH₃ 被 1 個 H 分裂

答案：4-isopropyltoluene (p-cymene)

結構：CH₃-C₆H₄-CH(CH₃)₂（甲基和異丙基在苯環 para 位）

HDI = (20+2-14)/2 = 4 = 苯環 ✓

高頻 C₈H₁₀O + 多種光譜 → 推結構

出現次數：110 Q10b、109 Q10b、107 小考 Q3

常見答案思路：

  • HDI = (16+2-10)/2 = 4 → 苯環
  • 看 IR 有沒有 broad O-H（~3400）→ alcohol
  • 看 IR 有沒有 C=O（~1700-1750）→ 若有則非 alcohol
  • 看 ¹H NMR 芳香區的 pattern → 判斷取代模式

題型九：DEPT + ¹³C NMR → 推結構（10 分）

高頻 給 ¹H NMR + ¹³C NMR 或 DEPT-135，推結構

出現次數：110 Q11、109 Q11、107 Q9

常見分子式：C₈H₁₀Br、C₆H₁₄O、C₈H₁₆O 等

DEPT-135 判讀：

  • 朝上 ↑ = CH₃ 和 CH
  • 朝下 ↓ = CH₂
  • 消失 = quaternary C（四級碳）

搭配 ¹³C 化學位移：

  • δ 0-50 → sp³ C（烷基）
  • δ 50-90 → C-O（接氧的碳）
  • δ 110-160 → sp² C（芳香/烯碳）
  • δ 160-220 → C=O

題型十：Oxymercuration-demercuration 機制（8 分）

必考 Alkene + 1) Hg(OAc)₂/H₂O 2) NaBH₄ → 寫機制

出現次數：110 Q12、109 Q12、107 Q12 → 三年都考！

機制步驟：

  1. Hg(OAc)₂ 的 Hg²⁺ 與 alkene 的 π 電子反應 → 形成 mercurinium ion（三員環，Hg 橋接兩個碳）
  2. H₂O（親核劑）攻擊 mercurinium ion → Markovnikov 方向（攻擊取代基較多的碳）→ anti addition
  3. 去質子化 → 得到含 Hg 的 alcohol 中間體
  4. NaBH₄ 還原 → 移除 Hg → 得到最終 Markovnikov alcohol

題目通常給內環 alkene（如 methylenecyclohexane 或 cyclopentene 衍生物），要求寫出兩個 regioisomer 產物 A 和 B。

題型十一：其他光譜推結構題（小考型）

高頻 ¹³C NMR assign peaks

出現：107 小考 Q1（assign ethyl benzoate 的 ¹³C 峰）

給一個已知結構的分子 + ¹³C NMR 光譜，要你把每個峰對應到分子中的哪個碳。

高頻 C₄H₈Cl₂ + ¹H NMR → 推結構

出現：107 小考 Q2

給化學位移和積分，推斷含兩個 Cl 的分子結構。注意 Cl 的 deshielding effect。

高頻 MS 區分異構物（如 methylcyclohexane vs ethylcyclopropane）

出現：107 小考 Q4

兩者分子量相同，但碎裂模式不同 → 看碎片差異判斷。

Ch10 Alkynes — 新增預測題（過去沒考過）

今年首次加入 Ch10，以下是根據講義重點預測的題目：

預測 1：Alkyne 還原 → 指定立體化學的 alkene

給一個 internal alkyne，用不同條件還原：

  • (a) H₂ / Lindlar's catalyst → cis (Z) alkene（syn addition）
  • (b) Na / NH₃(l) → trans (E) alkene（anti addition）
  • (c) excess H₂ / Pd → alkane

預測 2：Alkyne 水合 → Ketone 或 Aldehyde

  • H₂O / H₂SO₄ / HgSO₄ → Markovnikov → Ketone（經 enol → keto-enol tautomerism）
  • 1) R₂BH (bulky borane) 2) H₂O₂/NaOH → Anti-Markovnikov → Aldehyde

預測 3：Terminal alkyne alkylation 合成設計

從 acetylene 出發，合成特定的 internal alkyne：

  1. HC≡CH + NaNH₂ → HC≡C⁻ Na⁺
  2. + R₁-X (methyl 或 1° only!) → R₁-C≡CH
  3. + NaNH₂ → R₁-C≡C⁻
  4. + R₂-X → R₁-C≡C-R₂

注意：2° 和 3° RX 不行（會走 E2 消除）！

預測 4：Alkyne 的製備（Double dehydrohalogenation）

Vicinal 或 geminal dihalide + excess NaNH₂ → alkyne（兩次 E2）

若產生 terminal alkyne → 會被 NaNH₂ 去質子化 → 最後加 H₂O 得到 alkyne

預測 5：多步合成串聯（Ch10 + Ch13）

可能出現 alkyne → alkene → alcohol → ketone 的串聯合成：

  1. Terminal alkyne + NaNH₂ + R-X → internal alkyne（Ch10 alkylation）
  2. Internal alkyne + H₂/Lindlar → cis alkene（Ch10 reduction）
  3. Cis alkene + BH₃·THF + H₂O₂/NaOH → alcohol（Ch13 hydroboration）
  4. Alcohol + PCC → aldehyde 或 Na₂Cr₂O₇ → carboxylic acid（Ch13 oxidation）
