Chapter 16 — Nuclear Magnetic Resonance (NMR)

Part 2：¹H NMR + Spin-Spin Splitting + 猜題

「¹H NMR 是整張考卷的大魔王。化學位移、積分、分裂，三個東西搞懂就贏了。」——洛緋

¹H NMR 質子等價性 Proton Equivalence 超重要

判斷方法：把每個 H 分別替換成 X，看產生的化合物之間的關係：
替換結果名稱NMR 訊號
得到完全相同的分子 Homotopic（同位的） 相同訊號（永遠等價）
得到一對鏡像異構物（enantiomers） Enantiotopic（對映位的） 相同訊號（在無手性環境中）
得到一對非鏡像異構物（diastereomers） Diastereotopic（非對映位的） 不同訊號！（即使看起來很像）
得到不同的構造異構物（constitutional isomers） Constitutionally heterotopic（構造異位的） 不同訊號
最常考的陷阱：Diastereotopic protons！
當分子有 chiral center（手性中心）時，即使是同一個 CH₂ 上的兩個 H 也可能不等價 → 會出不同的 NMR 訊號 → 考試最愛考「這個分子有幾組不等價質子？」

16.6 積分 Integration 必考

Integration（積分）（訊號面積）：¹H NMR 中，訊號的面積比 = 質子數比

例：ethanol CH₃CH₂OH → 三組訊號，面積比 = 3 : 2 : 1（對應 CH₃ : CH₂ : OH）

注意：積分給的是比例，不是絕對數量！
看到比例 1.5 : 1.0 → 乘以 2 → 3 : 2
¹³C NMR 沒有可靠的積分！只有 ¹H NMR 的積分可以用來數質子。

16.7 自旋-自旋分裂 Spin-Spin Splitting 大魔王

Spin-spin splitting（自旋-自旋分裂 / 耦合）：鄰近碳上的不等價質子會讓訊號分裂成多個峰。

n+1 Rule：如果鄰近碳上有 n 個等價 H，訊號會被分裂成 n+1 個峰。
鄰近等價 H 數 (n)峰數 (n+1)名稱強度比（Pascal 三角形）
01Singlet (s) 單峰1
12Doublet (d) 雙峰1 : 1
23Triplet (t) 三重峰1 : 2 : 1
34Quartet (q) 四重峰1 : 3 : 3 : 1
45Quintet 五重峰1 : 4 : 6 : 4 : 1
56Sextet 六重峰1 : 5 : 10 : 10 : 5 : 1
Coupling constant (J)（耦合常數）：多重峰中，相鄰峰之間的距離（單位 Hz）。
互相耦合的兩組質子具有相同的 J 值！

三條重要規則

Rule 1：等價質子不會互相分裂
例：ethane CH₃-CH₃ → 所有 H 等價 → singlet

Rule 2：n+1 rule 只適用於鄰近碳上的 H
隔兩個碳以上的 H 通常不會造成分裂

Rule 3：分裂只發生在不等價質子之間
「看到 quartet 旁邊一定有 triplet。看到 triplet 旁邊一定有 quartet 或 doublet。它們是成對的。找到配對關係，結構就出來了。」
經典組合：
CH₃-CH₂-X 的 NMR：
• CH₃（3H）被旁邊 CH₂ 的 2 個 H 分裂 → triplet (t)
• CH₂（2H）被旁邊 CH₃ 的 3 個 H 分裂 → quartet (q)
→ 看到 t + q 的組合，積分比 3:2 → 幾乎確定有 ethyl group（-CH₂CH₃）！

¹H NMR 解題 SOP

Step 1：算 HDI（從分子式）→ 判斷有沒有環或不飽和鍵

Step 2：數峰數 → 幾組不等價質子

Step 3：看積分 → 每組多少個 H

Step 4：看化學位移 → 每組 H 的化學環境（sp³? 接 O? 芳香環?）

Step 5：看分裂模式 → n+1 rule 推斷鄰近 H 的數量

Step 6：拼結構 → 把碎片連起來，驗證是否符合所有數據

猜題 Ch16 Exam Predictions

猜題 1：判斷不等價質子數

畫出以下分子的 ¹H NMR 會有幾組訊號：

  • (a) CH₃OCH₃ → 1 組（所有 H 等價）→ singlet
  • (b) CH₃CH₂OCH₃ → 3 組（CH₃ 的 t, CH₂ 的 q, OCH₃ 的 s）
  • (c) para-xylene (1,4-dimethylbenzene) → 2 組（Ar-H 和 CH₃）

猜題 2：從 NMR 數據推結構（最常考的題型！）

分子式 C₄H₉Br，¹H NMR 數據：

  • δ 1.04 (d, 6H)
  • δ 1.95 (m, 1H)
  • δ 3.33 (d, 2H)

解法：

  1. HDI = (8+2-9-1)/2 = 0 → 無環無雙鍵
  2. δ 3.33 (d, 2H) → CH₂ 接 Br（downfield），被 1 個鄰近 H 分裂成 doublet
  3. δ 1.95 (m, 1H) → CH，被兩側的 H 分裂成 multiplet
  4. δ 1.04 (d, 6H) → 兩個等價 CH₃，被 1 個 H 分裂成 doublet
  5. 拼起來：(CH₃)₂CH-CH₂Br = isobutyl bromide（1-bromo-2-methylpropane）

猜題 3：Homotopic / Enantiotopic / Diastereotopic 判斷

判斷以下分子中標記的兩個 H 的關係：

  • (a) CH₂Cl₂ 中的兩個 H → 替換成 X 得到相同分子 → Homotopic
  • (b) CH₂ClBr 中的兩個 H → 替換成 X 得到一對 enantiomers → Enantiotopic
  • (c) 2-bromobutane 的 C3 上兩個 H → 替換成 X 得到一對 diastereomers → Diastereotopic（不同 NMR 訊號！）

猜題 4：¹³C NMR 峰數

以下分子在 ¹³C NMR 中各有幾個峰？

  • (a) cyclohexane → 1 個峰（所有碳等價）
  • (b) 1-chloro-methylcyclohexane → 利用對稱性數碳
  • (c) 2-butanone CH₃COCH₂CH₃ → 4 個峰（4 種不等價碳）

猜題 5：分裂模式配對

CH₃CH₂OH 的 ¹H NMR 中：

  • CH₃ → 被 CH₂ 的 2H 分裂 → triplet，δ ~1.2
  • CH₂ → 被 CH₃ 的 3H 分裂 → quartet，δ ~3.7（接 O 所以 downfield）
  • OH → 通常 broad singlet（因為快速交換，不被分裂）
  • 積分比：3 : 2 : 1

猜題 6：IR + MS + NMR 綜合推斷

分子式 C₈H₁₀O，IR: broad ~3400 cm⁻¹，¹H NMR：

  • δ 7.2 (s, 5H)
  • δ 3.65 (t, 2H)
  • δ 2.85 (t, 2H)
  • δ 1.8 (s, 1H, broad)

解法：

  1. HDI = (16+2-10)/2 = 4 → 苯環！
  2. IR broad 3400 → O-H（alcohol）
  3. δ 7.2 (s, 5H) → 單取代苯環（C₆H₅-）
  4. δ 3.65 (t, 2H) → -CH₂-OH（接 O → downfield，被鄰近 CH₂ 分裂成 t）
  5. δ 2.85 (t, 2H) → Ph-CH₂-（接苯環，被鄰近 CH₂ 分裂成 t）
  6. δ 1.8 (s, 1H, broad) → O-H
  7. 答案：C₆H₅CH₂CH₂OH = 2-phenylethanol
