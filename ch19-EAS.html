Chapter 19 — Electrophilic Aromatic Substitution

親電芳香取代 (SEAr) · 考前重點整理
19.1EAS 機制總覽（σ complex）
Electrophilic Aromatic Substitution (EAS, SEAr)（親電芳香取代）：苯環的 H 被親電試劑 E⁺ 取代。雖然苯有 π 電子，但不是加成（會破壞芳香性），而是取代（保留芳香性）。
EAS 兩步驟通則：addition → σ complex（速率決定）→ elimination（恢復芳香）
關鍵能量觀念：第一步（形成 σ complex）失去芳香性，能量上昇高；這是 EAS 的速率決定步驟。第二步是脫去 H⁺，回復芳香性，能量大幅下降，因此快速。整體 ΔG 取決於 σ complex 的穩定性。
19.2Halogenation 鹵化
鹵素條件路易士酸產生的親電試劑
Br₂FeBr₃（少量）FeBr₃Br⁺ ⋯ FeBr₄⁻（極化的 Br-Br）
Cl₂AlCl₃ 或 FeCl₃AlCl₃Cl⁺ ⋯ AlCl₄⁻
I₂需要氧化（Cu²⁺、HNO₃、過氧化物）—I⁺（從 I₂ 氧化而來）
F₂不能用—太活潑，會破壞芳香環
機制要點：Br₂ 本身不夠親電；FeBr₃ 把 Br-Br 鍵極化（Br…Br-FeBr₃），讓近端 Br 變得帶部分正電荷 → 苯環攻擊它 → σ complex → FeBr₄⁻ 帶走 H⁺ → 產物 + HBr + FeBr₃（催化劑再生）。
19.3Sulfonation 磺化
條件：fuming H₂SO₄（H₂SO₄ + SO₃）。親電試劑為 SO₃（或 H–SO₃⁺，更強）。產物：苯環 → Ar–SO₃H（aryl sulfonic acid）。
磺化是可逆的！
強酸（濃 H₂SO₄）→ 磺化方向：Ar-H → Ar-SO₃H
熱稀水溶液 → 脫磺化方向：Ar-SO₃H → Ar-H
這個可逆性讓 -SO₃H 可作為「暫時阻擋基（blocking group）」，先佔住一個位置，之後再脫除。
19.4Nitration 硝化 + 還原成胺
條件：HNO₃ + H₂SO₄。親電試劑為 NO₂⁺（nitronium ion）。

NO₂⁺ 的生成：HNO₃ + 2 H₂SO₄ → NO₂⁺ + H₃O⁺ + 2 HSO₄⁻

（H₂SO₄ 質子化 HNO₃ 的 OH，使其脫水變成 NO₂⁺）

2-Step 安裝胺基（–NH₂）

由於 NH₂ 太強的活化基，很容易被氧化或多取代 → 不直接用 NH₃。改用兩步：

  1. HNO₃ / H₂SO₄ → 安裝 –NO₂
  2. 還原（Sn/HCl 或 Zn/HCl 或 H₂/Pd 或 Fe/HCl）→ –NH₂
19.5Friedel-Crafts Alkylation 烷基化
條件：R–X + AlCl₃（路易士酸） → 苯環攻擊 R⁺。產物 Ar–R。

三大限制

限制說明
① 重排1° 烷基鹵會經氫化物/烷基遷移 → 3°/2° 碳陽離子。例：1° n-propyl Cl → 3° isopropyl 取代產物。用醯化（19.6）+ 還原可避免。
② 多烷基化烷基是活化基，安裝後苯環反應更快 → 容易發生第二次、第三次烷基化。
③ 不適用強去活化環當苯環上已有 –NO₂、–SO₃H、–COOH、–CN 等強去活化基，或胺基（會跟 AlCl₃ 配位）→ 反應不發生。
④ 需 sp³ R-Xaryl halide（Ar-X）和 vinyl halide（C=C-X）的碳陽離子太不穩定 → 無法使用。
19.6Friedel-Crafts Acylation 醯化
條件：R–C(=O)–Cl（醯氯）或 (RCO)₂O（酸酐）+ AlCl₃ → 苯環攻擊 acylium ion R-C≡O⁺。產物 Ar–C(=O)–R（aryl ketone）。

對比烷基化的優點

性質F-C 烷基化F-C 醯化
中間體R⁺ 碳陽離子（會重排）R-C≡O⁺ acylium ion（共振穩定，不重排）
產物對環的影響烷基是活化基 → 多烷基化羰基是去活化基 → 不會多醯化
合成策略：要安裝 1°、不重排的烷基時 → 先 F-C 醯化 → 再用 Clemmensen 還原（Zn(Hg) / HCl, heat）或 Wolff-Kishner（NH₂NH₂, KOH） 把 C=O 還原成 CH₂。等價於成功安裝直鏈烷基。
19.7-8活化/去活化基 + 區位選擇性
核心二刀流：
① 活化基（EDG, electron-donating） → 反應比苯快 → ortho/para director
② 去活化基（EWG, electron-withdrawing） → 反應比苯慢 → meta director
③ 鹵素例外 → 去活化但 ortho/para director（19.9）
類別典型基團機制導向
強活化–O⁻、–NH₂、–NHR、–NR₂、–OH、–OR共振給電子（孤對 → 環）o/p
中等活化–NHCOR、–OCOR共振 + 電負度的競爭，仍偏共振o/p
弱活化–R（烷基）、–Ar（苯基）誘導/超共軛o/p
弱去活化–F、–Cl、–Br、–I誘導吸電（去活）+ 共振給電子（o/p 導向）o/p（鹵素例外）
中等去活化–CHO、–COR、–COOR、–COOH、–CONH₂共振拉電子m
強去活化–NO₂、–SO₃H、–CF₃、–NR₃⁺、–CN共振 + 誘導都吸電m

為何 EDG → ortho/para？

當 E⁺ 攻擊 ortho 或 para 位置時，所形成的 σ complex 共振中，正電荷會落在連有 EDG 的碳上。EDG 的孤對電子可以「把正電中和掉」（額外共振式）→ σ complex 多一個共振結構 → 更穩定 → 反應快。

若攻擊 meta 位置，正電荷不會落在連 EDG 的碳上 → 沒有額外共振穩定 → 較慢。

為何 EWG → meta？

EWG 攻擊 ortho 或 para 時，正電荷會落在連有 EWG 的碳上 → 正電貼正電（超不穩定）。攻擊 meta 時則避開這個情況，正電在其他位置 → 較穩定 → meta 主導。

19.9Halogen Exception 鹵素例外
鹵素（F、Cl、Br、I）同時有兩種效應：
• 誘導吸電（電負度大）→ 整體去活化
• 共振給電子（孤對給入環）→ ortho/para 導向
結果：反應比苯慢，但定位於 ortho/para。獨一無二的「去活化但 o/p」。
19.10強度排序與判斷流程

判斷流程 SOP

  1. 看官能基直接連到苯環的第一個原子
  2. 該原子有孤對 + 沒帶正電 → EDG（共振給電子）→ ortho/para
  3. 該原子有正電（或 π 鍵連到電負原子） → EWG（共振拉電子）→ meta
  4. 該原子是 烷基/芳基 → 弱活化（超共軛）→ ortho/para
  5. 該原子是 鹵素 → 弱去活化但 ortho/para
19.11多取代基

① 加成性（addition of effects）

兩個取代基的導向效應「方向相同」時 → 結果加成（疊加），更專一。

② 競爭時的判定

方向不同時：
• 較強的活化基「贏」（決定主產物位置）
• 通常會得到混合產物

③ 阻擋技巧（blocking group）

想要 ortho 取代但 para 也活化時：先用磺化（–SO₃H）佔住 para 位置 → 反應只能 ortho → 最後熱稀水溶液脫去 –SO₃H → 完成 ortho 專一合成。

19.13SNAr 親核芳香取代
SNAr (addition-elimination)：當苯環上同時有離去基（如 Cl、F）與強 EWG 在 ortho/para 時，親核試劑 Nu⁻ 可以攻擊苯環 → 形成 Meisenheimer complex（負電σ加合物） → 離去基離去 → 取代。
SNAr 兩步驟：addition（形成 Meisenheimer 負離子）→ elimination（離去基離去）
SNAr 必要條件：
① 苯環上有 離去基（–Cl, –F, –Br, –NO₂...）
② 強 EWG（–NO₂、–CN、–CF₃...）位於 ortho 或 para（共振穩定 Meisenheimer 負電）
③ 強親核試劑（–OH、–OR、–NR₂、–CN...）
注意：F > Cl > Br > I（與 SN2/SN1 相反，因為這裡 C-F 鍵極化最強，addition 步是速率決定）
19.14Benzyne 苯炔中間體
當 Ar-X 沒有 EWG 活化時，強鹼（如 NaNH₂、NaOH 高溫高壓）可以走另一條路徑：脫 H + 脫 X（E2 機制）→ 形成benzyne（苯環中含三鍵） → 親核試劑加成。

機制

  1. elimination：強鹼拔走鄰位 H + X⁻ 離去 → 形成 benzyne（苯環中有 C≡C 三鍵）
  2. addition：Nu⁻ 攻擊 benzyne 任一邊（兩邊機率不同，看穩定性）
  3. 得到取代產物

區別 SNAr vs Benzyne：
• SNAr：需要 EWG 活化，取代位置 = 原離去基位置
• Benzyne：不需 EWG，取代位置可能在原位置或鄰位（兩種產物）

Benzyne 的證據：讓 benzyne 與 Diels-Alder 試劑（如 furan、anthracene）反應 → 得到加成環產物，證明 benzyne 中間體真實存在。

目標基試劑

19.47 — 二硝化（聯苯醚 + sec-butyl）
給定化合物（兩個苯環中間有 –O– 連結，每環各有 sec-butyl 基）+ HNO₃ / H₂SO₄ / 50°C → 兩個 –NO₂ 取代產物，畫出結構。
Treatment with HNO₃/H₂SO₄ at 50°C affords a dinitration product. Draw it.

分析步驟

  1. 每個苯環有 2 個取代基：–OR（中等活化、o/p 導向）、–sec-butyl（弱活化、o/p 導向）
  2. 方向相同（都 o/p） → 較強的（–OR）控制
  3. –OR 的 ortho 一邊已被 sec-butyl 佔住 → 剩另一個 ortho 與 para 可硝化
  4. 該另一個 ortho 也立體位阻過大（緊貼 sec-butyl）→ 實際只在 另一個 unencumbered 位置（多半是另一 ortho 或 para）發生硝化
  5. 每環各裝一個 –NO₂ → 共 2 個
雙取代基同向時，由「較強的活化基」+「立體位阻」共同決定最終位置。
19.51 — SNAr 機制（Ar–Cl → Ar–OEt）
提出機制：芳香化合物（含 Cl 與羰基 ortho/para）+ NaOEt → 取代 Cl 為 OEt + NaCl。
Propose a plausible mechanism for the SNAr transformation.

機制（addition-elimination）

  1. Step 1：OEt⁻ 親核攻擊連 Cl 的碳 → 形成 Meisenheimer complex（負電σ加合物）
  2. 負電靠 ortho/para 的羰基（–C=O）共振穩定 → 中間體可分離
  3. Step 2：Cl⁻ 離去，重建芳香性 → Ar–OEt 產物

關鍵：羰基 ortho 或 para 是必要的，能把負電從環吸到羰基 O 上（共振）。若羰基在 meta，就不能穩定 Meisenheimer 負電 → 不反應。

19.52 — 預測產物
預測下列各反應的產物，含區位選擇性。
Predict the product(s) of the following reactions.

(a) p-溴甲苯 → 1) HNO₃/H₂SO₄, 2) Zn/HCl, 3) NaOH

–CH₃（弱活化、o/p）vs –Br（弱去活化、o/p）→ CH₃ 較強 → 由 CH₃ 主導。CH₃ 的 para 已被 Br 佔住 → 走 ortho（相對於 CH₃）。

步驟 2-3：–NO₂ → –NH₂（還原 + 鹼處理）

產物：4-溴-3-氨基甲苯（4-bromo-3-aminotoluene 或 5-bromo-2-aminotoluene 同義）

(b) p-xylene → 1) AlCl₃ + EtCH₂COCl, 2) Zn(Hg)/HCl, heat

p-xylene 兩個 –CH₃ 對稱 → 4 個剩餘位置等價（C2=C3=C5=C6）→ 第三取代只有一個區位結果。

步驟 1：F-C 醯化裝 –CO–CH₂CH₃。步驟 2：Clemmensen 還原 → –CH₂CH₂CH₃（n-propyl）。

產物：1-propyl-2,5-dimethylbenzene（不重排，因為走醯化先）

(c) 異丙基苯 → 1) CH₃Cl/AlCl₃, 2) KMnO₄/H₂O/heat, 3) H₃O⁺

–CH(CH₃)₂ ortho 位置立體位阻大 → para 主導。步驟 1 裝 –CH₃ 在 para。步驟 2-3：兩個 alkyl 都被 KMnO₄ 氧化成 –COOH（苄位完全氧化）。

產物：對苯二甲酸 terephthalic acid（HOOC-C₆H₄-COOH, p-）

(d) tert-butylbenzene → 1) CH₃Cl/AlCl₃, 2) excess NBS/heat

–C(CH₃)₃ ortho 立體位阻極大 → para 主導。步驟 1 裝 –CH₃ 在 para。步驟 2：NBS 苄位溴化，tert-butyl 沒苄位 H（其碳全 CH₃）→ 不被溴化；只 –CH₃ 被反覆溴化 → –CBr₃。

產物：4-tert-butyl-(tribromomethyl)benzene

19.54 — 找合成漏洞
下列各合成不會給出預期產物。指出每個的問題。
Identify the flaw in each synthesis.
  • (a) 先硝化再 F-C 烷基化 → 失敗：硝基苯是強去活化環，不能進行 F-C 反應。
  • (b) 用 1-氯丙烷 + AlCl₃ 想裝 n-propyl → 失敗：1° 烷基會重排成 isopropyl（2°）。應改走醯化 + Clemmensen。
  • (c) 想在 m 位置裝醯基，但起始物的取代基是 o/p 導向 → 醯基會去 ortho/para 而不是 meta。需重新設計順序。
  • (d) 想在 ortho 位置裝溴，但起始物有 tert-butyl（巨大基）→ ortho 立體位阻過大 → 反應主走 para。需改用阻擋技巧。
合成失敗的常見原因：① F-C 在去活化環上、② 1° 烷基重排、③ 取代基順序錯（導向不符）、④ 立體位阻把 ortho 擋掉。
19.58 — 二醇環化（雙重 F-C 烷基化）
苯 + 二醇（HO–CMe₂–CH₂CH₂–CMe₂–OH）+ H₂SO₄ → 1,1,4,4-tetramethyltetralin（六員環稠合在苯上）。提出機制。
Propose a mechanism for the cyclization to tetralin.

機制（雙重 F-C 級聯）

  1. H₂SO₄ 質子化二醇的一個 –OH → –OH₂⁺
  2. 失水（H₂O 離開）→ 3° 碳陽離子（tert-carbocation）（穩定）
  3. 苯環攻擊此 carbocation → σ complex → 失 H⁺ → 第一次烷基化完成（單環產物）
  4. 另一邊 –OH 也質子化 + 失水 → 第二個 3° 碳陽離子
  5. 此時苯環已連接，分子內攻擊新形成的 carbocation → 形成新六員環（稠合於苯）→ 失 H⁺
  6. 產物：1,1,4,4-tetramethyl-1,2,3,4-tetrahydronaphthalene
兩個 –OH 在質子化下變成暫時的 carbocation 親電試劑（等同於 F-C 烷基化），且因 3° 碳陽離子很穩定，反應順利。第二步是分子內的，所以效率高。
19.62 — 辨識 A 與 B（單溴化產物數目差異）
A 與 B 都是芳香酯 C₈H₈O₂。Br₂/FeBr₃ 處理：A 給1 個單溴產物、B 給2 個單溴產物。判斷 A 與 B 的結構。
A and B are aromatic esters C₈H₈O₂. A → 1 monobromination product; B → 2 products. Identify A and B.

不飽和度 = (2×8+2−8)/2 = 5 → 苯環(4) + C=O(1) ✓

2 個 ester 結構的可能：

  • A = methyl benzoate（Ph–C(=O)–O–CH₃）：羰基直接連環 → 是 meta director → 兩個 meta 等價 → 1 個產物。
  • B = phenyl acetate（Ph–O–C(=O)–CH₃）：氧直接連環 → 是 o/p director（共振給電子） → ortho 與 para 不同 → 2 個產物。

關鍵：看「直接連到苯環的原子是 C 還是 O」。連 C（且 C 帶 π=O）→ 拉電子（meta director）；連 O（孤對給入環）→ 給電子（o/p director）。

19.67 — 多步合成（多種多取代苯）
從苯出發合成多種多取代苯化合物（多題）。
Design syntheses for various polysubstituted benzenes starting from benzene.

關鍵策略

  • 順序最重要：先裝 m-導向基（–NO₂、–COR、–COOH）→ 後裝其他基（會去 m）。或反過來。
  • F-C 不能在強去活化環上：所以醯化/烷基化要先做（在還沒裝 NO₂ 之前）。
  • 1° 烷基要走醯化 + Clemmensen，避免重排。
  • 需要 –NH₂：先裝 –NO₂，最後一步還原成 –NH₂。
  • 需要 ortho 但 para 被佔：用 –SO₃H 阻擋 para → 反應 → 脫 –SO₃H。
  • 立體位阻：bulky 基的 ortho 通常打不到，預期 para 主導。
多步合成的逆推：從產物的取代基模式判斷導向順序，並排除 F-C 在去活化環上、1° 重排等陷阱。
19.91 — ICl 親電碘化（區位選擇性）
給定化合物 + ICl → 兩個產物 A:B = 3:1。提出機制並解釋為何 A 較主導。
ICl serves as I⁺ source. Explain why A:B = 3:1.

機制：I–Cl 極化（Cl 較電負），等效 I⁺ 攻擊苯環 → σ complex → 失 H⁺ → Ar–I + HCl。

(a) A 與 B 的差別：主產物 A 的新 C–C（或 C–I）鍵在立體位阻較小、或共振穩定 σ complex 較多的位置。

(b) 為何 A > B：A 對應的 σ complex 中，正電可以被取代基的孤對/π 系統多重共振穩定；B 的位置則只能受少數共振穩定 → A 較穩定 → 反應更快。

19.93 — B–N 多環芳香（分子內 EAS）
合成中以 B–N 取代 C=C 的多環芳香 3，從化合物 1 出發：步驟 1) n-BuLi，步驟 2) BCl₃ → 中間體 2 → AlCl₃ → 化合物 3。提出機制。
Polycyclic with B–N replacing C=C, made via two consecutive intramolecular EAS.

機制（兩次 EAS 級聯）

  1. 步驟 1（1 → 2）：n-BuLi 拔走 N–H 質子 → N⁻ 親核攻擊 BCl₃ → 形成 B–N 鍵 + 釋放 HCl；BCl₃ 還剩 2 個 Cl，作為親電中心。
  2. 步驟 2（2 → 3）：AlCl₃ 拔走 B 上的一個 Cl → B 變成更強的親電中心 → 苯環親核攻擊 B → σ complex → 失 H⁺ → 第一個新 B–C 鍵。重複一次 → 第二個新 B–C 鍵 → 形成稠合多環骨架 3。

兩步驟都是分子內 EAS，新形成的 σ 鍵 = B 與苯環碳。AlCl₃ 為路易士酸催化（活化 B-Cl）。

整章串聯記憶：
① EAS = addition (slow, σ complex) + elimination (fast, lose H⁺)
② 5 個基本反應：halogenation、sulfonation、nitration、F-C alkylation、F-C acylation
③ EDG → o/p、EWG → m、鹵素 = 去活化但 o/p（例外）
④ 取代基順序是合成的靈魂：m-導向基先做、F-C 不能在去活化環上、1° 烷基要走醯化+Clemmensen
⑤ 阻擋技巧：用磺化暫時佔位
⑥ SNAr：要 LG + EWG 在 o/p；走 Meisenheimer 中間體；F > Cl > Br > I
⑦ Benzyne：不需 EWG，強鹼 + Ar-X，產物可能在原位置或鄰位
