Chapter 16 — Nuclear Magnetic Resonance (NMR)

Part 1：基本原理 + ¹³C NMR

「NMR 就是把分子丟進磁場裡，聽每顆原子核在不同頻率上唱歌。頻率不同，位置就不同。」——洛緋

16.1 NMR 基本原理

Nuclear Magnetic Resonance (NMR)（核磁共振）：有機化學中最有價值的結構鑑定工具。
原理：某些原子核有自旋（spin）→ 在外加磁場中，自旋可以與磁場平行（低能量）或反平行（高能量）→ 吸收特定頻率的無線電波（radio frequency）→ 翻轉到高能態 = resonance（共振）

哪些核有 NMR 活性？

Spin number (I)條件範例NMR 活性？
Half-integer (½)質量數為奇數¹H, ¹³C, ¹⁹F, ¹⁵N, ³¹P有（最常用）
Integer (1)質量數偶、原子序奇²H, ¹⁴N有（但較少用）
Zero (0)質量數偶、原子序偶¹²C, ¹⁶O無！
¹²C 和 ¹⁶O 沒有 NMR 訊號！
所以 ¹³C NMR 靠的是天然豐度只有 1.1% 的 ¹³C → 需要大量累積訊號（FT-NMR）
ΔE = hν = γħB₀
γ (gyromagnetic ratio 磁旋比)：每種核固有的常數
B₀：外加磁場強度 → 磁場越強 → 能量差越大 → 解析度越好

常見磁場：7.04 T（對應 ¹H 300 MHz）、9.4 T（對應 ¹H 400 MHz）
Shielding（屏蔽）（遮蔽效應）：鍵中的電子會產生一個小磁場，抵消外加磁場 → 核感受到的有效磁場較弱 → 需要更強的磁場才能共振 → 訊號出現在 upfield（高場，右邊）

Deshielding（去屏蔽）（去遮蔽效應）：附近有拉電子基 → 電子密度降低 → 屏蔽減少 → 訊號出現在 downfield（低場，左邊）

16.5 化學位移 Chemical Shift 核心概念

Chemical shift (δ)（化學位移）：以 ppm (parts per million) 為單位，衡量某核相對於參考標準的共振位置。

TMS (Tetramethylsilane, (CH₃)₄Si)（四甲基矽烷）：定義為 δ = 0 ppm。Si 的電正度低 → 碳上電子密度高 → 高度屏蔽 → 幾乎所有有機化合物訊號都在 TMS 的左邊（downfield）。

δ = (頻率差 / 儀器頻率) × 10⁶
例：400 Hz 的差異在 400 MHz 儀器上 = 1 ppm → ppm 值與儀器頻率無關（不同儀器得到相同 δ）
NMR 光譜方向：

← Downfield（低場） ——————————— Upfield（高場） →
← Deshielded（去屏蔽） ———————— Shielded（屏蔽） →
← δ 大（10 ppm）—————————————— δ 小（0 ppm, TMS）→

¹³C NMR

¹³C NMR 的化學位移範圍：δ 0 ~ 220 ppm（比 ¹H 寬很多）

FT-NMR (Fourier Transform NMR)（傅立葉轉換核磁共振）：用寬頻脈衝一次激發所有 ¹³C → 收集 FID（free induction decay）→ 傅立葉轉換成頻譜 → 重複累積以消除雜訊

¹³C 化學位移範圍 必背

碳的類型δ 範圍 (ppm)說明
sp³ C（烷基碳）0 ~ 90無雜原子時 0~50；接 O/N/鹵素時往 downfield 移
sp² C（烯碳、芳香碳）110 ~ 160
C=O（羰基碳）160 ~ 220最 downfield！酮 ~200-210、醛 ~190-200、酯 ~170、羧酸 ~175
¹³C NMR 看什麼？
→ 有幾個峰 = 有幾種不等價碳（nonequivalent carbons）
→ 峰的位置判斷碳的類型（sp³ / sp² / C=O）
→ 注意：¹³C NMR 的峰高度不代表碳的數量（跟 ¹H 不同！）

DEPT NMR

DEPT (Distortionless Enhancement by Polarization Transfer)（無失真極化轉移增強）：
可以區分不同類型的碳：

DEPT-135：
• CH₃ 和 CH → 朝上的峰（positive）
• CH₂ → 朝下的峰（negative）
• Quaternary C（四級碳）→ 不出峰

搭配 broadband decoupled spectrum（顯示所有碳）對比 → 消失的峰就是四級碳
「DEPT-135：上面是 CH₃ 跟 CH，下面是 CH₂，消失的是四級碳。分三類，很直覺。」
