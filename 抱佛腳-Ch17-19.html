2.1 芳香性判別 SOP 6-8 分 ★★
給你一個結構，問是 aromatic / anti-aromatic / non-aromatic / aliphatic？
4 步驟流程：① 是不是環？② 是不是平面？③ 環中所有原子有 p 軌域？④ 數 πe⁻ 套 Hückel。
📷 看原題（110 學年度 Q5：4 個結構判別）
110 學年度 Q5（8 分）— 4 個結構分別判 aromatic / anti-aromatic / non-aromatic / aliphatic。同頁含 Q3 預測產物 + Q4 IUPAC 命名。
類別定義穩定性典型例子
aromatic
芳香		 ① 環狀 ② 平面 ③ 環中每個原子皆 sp²（含 p 軌域，可形成連續 π 系統） ④ πe⁻ = 4n+2（2, 6, 10, 14…） 超穩定
（共振能 ~150 kJ/mol）		 苯（6π）、pyridine、pyrrole、cyclopentadienyl anion、tropylium cation、naphthalene、pyrene
anti-aromatic
反芳香		 ① 環狀 ② 平面 ③ 環中每個原子皆 sp² ④ πe⁻ = 4n（4, 8, 12…） 非常不穩定
（比一般共軛二烯還糟，能避就避）		 cyclobutadiene（4π）、cyclopentadienyl cation（4π）、cyclopropenyl anion
non-aromatic
非芳香		 是環，但任一條件不滿足：
• 環中有 sp³ 原子（破壞 p 軌域連續性）
• 環不平面（如 8 員環 tub-shape）
• 雖滿足 aromaticity 條件但結構扭曲變不平面以避免 anti-aromaticity		 跟一般烯類差不多，沒有特別穩或不穩 cyclohexane（全 sp³）、1,3-cyclohexadiene（有 sp³ C）、cyclooctatetraene（8π 但 tub）、N-methylpyrrolium（環有 sp³ C）
aliphatic
脂肪族		 不是環 — 正己烷、1,3-丁二烯、丙酮
判別 SOP（4 步驟）：
  1. 不是環？ → aliphatic
  2. 是環，但環中有 sp³ C/N？（如 N-methylpyrrolium 的 -CH₂-，cyclohexane 的所有 C）→ non-aromatic
  3. 是環、平面、全 sp²（或含 p 軌域）？數 πe⁻：
    • = 4n+2 → aromatic
    • = 4n → anti-aromatic
  4. 大環（≥8 員）？就算 πe⁻ 對，也要看是否真的能維持平面。8 員 cyclooctatetraene 雖 8π（4n）「應該」 anti-aromatic，但它扭成 tub 形避免了 anti-aromaticity → non-aromatic
★ 數 πe⁻ 完整守則
  • 每個 C=C（或 C=N）在環中 → +2 πe⁻
  • Carbocation（C⁺）在環中 → 0 πe⁻（空 p 軌域，提供位置但無電子）
  • Carbanion（C⁻）在環中 → +2 πe⁻
  • 環上 N 的 lone pair：看 lone pair 在哪個軌域
    • 若 N 沒接 H（如 pyridine 的 N）→ lone pair 在 sp² 平面方向，不參與環內 π，貢獻 0
    • 若 N 接 H（如 pyrrole 的 N-H）→ lone pair 在 p 軌域，參與環內 π，貢獻 +2
  • 環上 O / S 的 lone pair：(furan、thiophene) 一個 lone pair 在 p 軌域貢獻 +2，另一在平面方向不算
經典坑：cyclopentadienyl 系列
  • Cyclopentadiene（中性，有 sp³ CH₂）：環有 sp³ C → non-aromatic
  • Cyclopentadienyl cation（C⁺，4π）：4π = 4n → anti-aromatic（**非常不穩定**）
  • Cyclopentadienyl anion（C⁻，6π）：6π = 4n+2 → aromatic（**超穩定**，這就是 cyclopentadiene pKa~16 的原因）

同一個 5 環，cation/中性/anion 三種狀態三種類別都不一樣！考試愛考。

數 πe⁻ 怎麼數：
  • 每個 C=C → 2 πe⁻
  • Carbocation（C⁺）在環中 → 0 πe⁻（空 p 軌域）
  • Carbanion（C⁻）/ N、O lone pair 在 p 軌域 → 2 πe⁻
  • 例如 pyrrole（NH）：2 個 C=C + N 的 lone pair = 4 + 2 = 6 πe⁻ → aromatic

2.2 pKa 比較（用芳香性解釋） 4-6 分 ★★
題目：兩個分子哪個比較酸？解釋原因。

📷 看原題（110 學年度 Q2：cycloheptatriene vs cyclopentadiene）
110 Q2（4 分）— pKa=16 vs pKa=36 哪個比較酸？答案：cyclopentadiene 較酸（失 H⁺ 後 6πe⁻ aromatic 穩定）。
核心觀念：
  • 失質子後若得到 aromatic 物種 → 此酸 強（低 pKa）
  • 失質子後若得到 anti-aromatic → 此酸 弱（高 pKa）
111 Q2 — Pyrrolium NH⁺（pKa=0.4） vs Pyridinium NH⁺（pKa=5.2）

哪個比較酸？A（pyrrolium）較酸

  • A（pyrrolium NH⁺）：N 的 lone pair 原本參與芳香性。質子化後 lone pair 拿去和 H 結合 → 環只剩 4 πe⁻ → anti-aromatic → 不穩定 → 容易失質子 → 較酸
  • B（pyridinium NH⁺）：N 的 lone pair 在 sp² 軌域（不參與環內 π）。質子化不影響芳香性 → 6 πe⁻ aromatic 保留 → 穩定 → 較不酸

3.3 從苯出發合成 — SO₃H 阻擋法 6-10 分 ★
苯 → ortho-tert-butyl-bromobenzene。關鍵：用 SO₃H 當 para-blocking group 強迫 Br 進 ortho，最後再去保護。
📷 看原題（110 學年度 Q9：苯 → o-tert-butylbromobenzene）
110 Q9（6 分）— 從 benzene 設計合成路線到 o-tBu-bromobenzene（標 major）。同頁含 Q10 雙苯酮 H₃O⁺ 重排機制（10 分大題）。
  1. 苯 + tBuCl / AlCl₃ → tert-butylbenzene（Friedel-Crafts，3° 不重排）
  2. + fuming H₂SO₄ / ice bath → 4-tBu-benzenesulfonic acid（SO₃H 進 para 阻擋）
  3. + Br₂ / FeBr₃ → 2-Br-4-tBu-benzenesulfonic acid（Br 被擠到 ortho）
  4. + dilute H₂SO₄, Δ → 移除 SO₃H → o-tBu-bromobenzene
什麼時候用 acylation 而不是 alkylation？
Alkylation 容易碳重排（1°→3°）。Acylation（acyl chloride）的 acylium ion 是 sp² 共振穩定，不會重排。先 acylation 接 ketone，再 Clemmensen 還原成 CH₂R 即得相同 alkyl 基但無重排問題。
3.4 SNAr Meisenheimer 機制 6 分 ★
芳環有 NO₂（強 EWG）+ Cl（離去基）+ 親核劑 → 取代。經 addition → elimination（Meisenheimer complex）。
NO₂ 必須在離去基的 ortho 或 para，才能共振穩定中間體。
📷 看原題（110 學年度 Q13：SNAr 主產物 + 機制）
110 Q13（6 分）— 3,4-dichloro-1-nitrobenzene + Na-C≡C-(p-OMe-Ph) → (a) 主產物結構 (b) 機制。答案：alkyne 取代 NO₂ 的 ortho 那個 Cl（共振穩定中間體）。
三步機制：
  1. Addition：親核劑攻擊 ipso C（Cl 接的那個 C）→ sp³ 中間體（Meisenheimer complex），負電荷在環上
  2. 共振穩定：負電荷透過共振分散到 ortho/para 的 NO₂ 上
  3. Elimination：Cl⁻ 離開 → 重建芳香性 → 主產物
110 Q13 / 111 Q13 3,4-dichloro-1-nitrobenzene + Na-C≡C-(p-OMe-Ph) → 主產物。
  • 有兩個 Cl：一個在 NO₂ 的 ortho（位置 2 或 4）、一個在 NO₂ 的 meta
  • 取代發生在 ortho 那個 Cl（共振可達 NO₂）
  • 主產物：alkyne 取代 NO₂ ortho 的 Cl

★ 進考場前 5 分鐘 cheat sheet ★

1. NMR 三題（直接背）

  • C₅H₁₀O = 2-pentanone（CH₃COCH₂CH₂CH₃）
  • C₉H₁₀O = propiophenone（PhCOCH₂CH₃）
  • C₃H₆O = propylene oxide（methyloxirane）

2. UV vs Heat 表（背口訣）

  • 「熱大光小」：heat con/dis 與 πe⁻ 同奇偶
  • 4n + heat = con；4n + UV = dis
  • 4n+2 + heat = dis；4n+2 + UV = con（Vitamin D 答案）

3. 名詞解釋必背 5 個

  • Claisen / Diels-Alder / SNAr / Clemmensen / Birch

4. 芳香性 4 步驟

  • ① 環？② 平面？③ 全 sp²？④ 4n+2 (aromatic) / 4n (anti)
  • 失質子後 aromatic 那個 → 較酸（pKa 低）

5. EAS 重排陷阱

  • Isobutyl-Cl + AlCl₃ → 主產物是 tert-butyl 不是 isobutyl（1°→3° 重排）
  • OH/NH₂ → ortho/para；NO₂/COR/COOH → meta

6. 苯合成 SO₃H 套路

  • tBu → 加 fuming H₂SO₄（para 鎖定）→ Br₂（強迫 ortho）→ dilute H₂SO₄ Δ（去保護）

7. SNAr 必背

  • NO₂ 必須在離去基 ortho/para 才會走 SNAr
  • 機制：addition → Meisenheimer → elimination

8. Vitamin D = 6π conrotatory（hν）

  • 機制畫 6 個 π 電子箭頭，標 conrotatory，最終 trans triene
