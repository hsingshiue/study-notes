題型二：pKa 比較 → 用芳香性解釋（4-6 分）
必考 兩個分子的 pKa 差異 → 解釋原因

出現次數：110 Q2、111 Q2 → 兩年都考

110 題目：Cycloheptatriene-H（pKa=16） vs Cyclopentadiene-H（pKa=36）

  • Cyclopentadiene 失 H⁺ → cyclopentadienyl anion 6πe⁻（4n+2）→ aromatic → 非常穩定 → 容易失質子（pKa 16，意外的酸）
  • 等等！題目寫 cyclopentadiene pKa=36 是反過來記憶 — 一般 sp³ C-H 才 pKa ≈ 50。Klein 的版本可能依比較條件不同，請以講義為準
  • 核心觀念：「失質子後若得到 aromatic 物種 → 強酸（低 pKa）」「若 anti-aromatic → 弱酸（高 pKa）」
記住數 πe⁻ → 對 4n+2 / 4n 公式，這題答的不是化學位移，是「失質子後的穩定性」。

111 題目：Pyrrolium-NH⁺（pKa=0.4） vs Pyridinium-NH⁺（pKa=5.2）

  • A（pyrrolium）pKa=0.4 ←較酸：N 上的 lone pair 原本參與芳香性（占 6πe⁻ 的 2 個）。質子化後 lone pair 拿去和 H 結合 → 環只剩 4πe⁻ → anti-aromatic / non-aromatic → 不穩定 → 容易失質子。
  • B（pyridinium）pKa=5.2：N 上的 lone pair 在 sp² 軌域上（不參與環內 π）。質子化不影響芳香性 → 6πe⁻ aromatic 保留 → 較穩定 → 較不容易失質子。
  • 結論：A 較酸（pyrrolium）。
解題 SOP：1) 畫脫質子（或加質子）後的物種 2) 數環內 πe⁻ 3) 套 4n+2（aromatic 穩）/ 4n（anti-aromatic 不穩）4) 寫穩定性比較。

題型五：Aromatic / Anti-aromatic / Non-aromatic / Aliphatic 判別（6-8 分）
必考 給多個結構判別芳香性類別

出現次數：110 Q5、111 Q5 → 兩年都考

判別流程（SOP）

  1. 是否為環？沒有環 → aliphatic
  2. 是否平面 + 環中所有原子 sp²/sp 含 p 軌域？有 sp³ 碳在環中（破壞共平面）→ non-aromatic
  3. 計算 πe⁻（雙鍵 ×2、carbocation 0、carbanion / 中性 lone pair 在 p 軌域 +2）：
    • 4n+2（2, 6, 10, 14...）→ aromatic
    • 4n（4, 8, 12...）→ anti-aromatic

反應本質：6π electrocyclic ring opening（B 環開環變 secosteroid）

立體化學規則表

Vitamin D 是 6πe⁻ cyclohexadiene 系統。Vitamin D₃ 是天然產物（光化學產物）→ 6π 系統 + UV → conrotatory。題目若是 thermal → disrotatory。

機制畫法要點

  1. 畫出原始 cyclohexadiene 的 6πe⁻ 系統（B 環）
  2. 標明三個雙鍵電子對同向旋轉（con）或反向（dis）
  3. 新形成的雙鍵端 H 朝向（cis or trans）來自原本的 sp³ 碳取代基
  4. 產物 = triene，secosteroid 的「鏡頭打開」狀態
考試的 Vitamin D₃ 產物 H 是 cis（朝同側）→ 對應光化學 conrotatory。
題型七：Sigmatropic 機制題（8 分）
必考 含 NTs 與 NMe₂-aryl 的多步 sigmatropic（135°C）

出現次數：110 Q7、111 Q7 → 兩年同題

反應步驟拆解

  1. Step 1：[3,3]-sigmatropic（aryl Cope/Claisen 式）
    • NMe₂-aryl 的鍵把 σ 鍵移位 → 形成 dienone-like 中間體（破壞苯環芳香性，但暫時的）
  2. Step 2：Intramolecular Diels-Alder [4+2]
    • 第一步形成的 diene + 鏈尾 allyl-NTs 雙鍵（dienophile）→ 環化 → bicyclic 產物
  3. 苯環芳香性恢復 → 得到帶 NMe₂ 與 NTs 取代的橋環產物

機制要點 ★

  • 每一步畫六員環椅式過渡態
  • 每個箭頭代表 2 個 π 電子的移動
  • 整體 π 電子數守恆（sigmatropic）
  • 產物的橋頭 H 立體：根據 supra/supra 或 supra/antara 決定
學生答案版（110 Q7）顯示的是「Cope-like → Diels-Alder」串聯。記得 [3,3] 步驟用 6π 電子的椅式 TS。

題型十：機制題（6-12 分）
必考 SNAr 機制（111 Q13、110 Q13）

反應：3,4-dichloro-1-nitrobenzene + Na-C≡C-(p-OMe-Ph) → major product

機制三步

  1. Addition：炔基陰離子（軟親核）攻擊 ipso to Cl（與 NO₂ 相鄰）→ 形成 Meisenheimer complex（sp³ 碳，負電在環上）
  2. 共振穩定：負電荷透過共振分散到 ortho/para 的 NO₂（這就是為何 NO₂ 必須在 ortho/para 位）
  3. Elimination：離去基（Cl⁻）離開 → 重建芳香性 → 主產物
主產物 = alkyne 取代原本 Cl 位置（NO₂ 的 ortho），因為這個位置的 Cl 離去後，負電荷可由 NO₂ 共振穩定。另一個 Cl（meta to NO₂）不容易被取代。
必考 Pinacol 重排（107 Q6b，Mcmurry）

反應：vicinal diol + H₃O⁺ → ketone（炔生產物）

  1. 一個 -OH 被質子化 → -OH₂⁺ → 失水形成 carbocation（選最穩定的位置：3° > 2° > 1°，aryl/allyl 加分）
  2. 鄰近碳的取代基（aryl > H ≈ alkyl）1,2-migration → 新 carbocation（α to OH）
  3. 剩下的 OH 失質子 → 形成 C=O ketone
高頻 POCl₃ / Pyridine 脫水（107 Q6a，Mcmurry）
  1. -OH 攻擊 POCl₂(=O)Cl → 生成 R-OPOCl₂（O 失質子，pyridine 接 H⁺）
  2. -OPOCl₂ 是很好的離去基（比 -OH 強多了）
  3. E2 反鍵消去（pyridine 拔 β-H）→ alkene

對比 H₂SO₄ 脫水（E1）：POCl₃/pyridine 走 E2，較不重排，立體受控。

高頻 I-Cl 親電加成到 alkyne（110 Q11）
  1. I-Cl 極化（I 帶 δ⁺，Cl 帶 δ⁻）
  2. I⁺ 攻擊 alkyne π 鍵 → iodiranium / vinyl cation 中間體
  3. Cl⁻ 從反面攻擊（anti addition）→ trans-haloalkene
  4. Regiochemistry：I 接到能讓 vinyl cation 更穩定的那個碳（Markovnikov 方向，aryl 一側更穩）
參考 Wittig 機制（107 Q8，Mcmurry）
  1. Ph₃P + R-CH₂-Br → Ph₃P⁺-CH₂-R Br⁻（phosphonium salt，SN2）
  2. BuLi 拔 α-H → ylide（Ph₃P=CHR ↔ Ph₃P⁺-CHR⁻）
  3. Ylide 的 C⁻ 攻擊 ketone C=O → betaine
  4. betaine 環化 → oxaphosphetane（4 員環 P-C-C-O）
  5. retro-[2+2] → alkene + Ph₃P=O
