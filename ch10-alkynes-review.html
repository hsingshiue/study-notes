Chapter 10 — Alkynes 炔類

考前重點整理 ⸜( ˶'ᵕ'˶)⸝

10.1 基本概念與用途

Alkyne（炔類）：含有 C≡C 三鍵的有機化合物，具有兩個 π 鍵，可作為親核劑（nucleophile）。
Acetylene（乙炔 C₂H₂）：最簡單的炔類，用於吹管焊接（blow torch）及作為合成前驅物。
補充：電石（CaC₂）加水產生乙炔 → CaC₂ + 2H₂O → C₂H₂↑ + Ca(OH)₂
傳統上用來催熟水果，但有雜質風險（砷、磷化物），不合法。真正的催熟激素是 Ethylene（乙烯）。

10.2 命名法 Nomenclature

  • IUPAC：將字尾 -ane 改為 -yne，三鍵給最低編號
  • 常用名：以 acetylene 為母體名（如 methylacetylene）
Terminal alkyne（末端炔）：三鍵在碳鏈末端，有 ≡C-H
Internal alkyne（內部炔）：三鍵在碳鏈中間
En-yne（烯炔）：同時含有雙鍵和三鍵的化合物

10.3 酸性 Acidity

Terminal alkyne 的 pKa ≈ 25
比 alkene（~44）和 alkane（~50）酸性強很多，因為 sp 碳的 s 軌域比例最高（50%），電子更靠近原子核，更能穩定負電荷。
能去質子化 terminal alkyne 的鹼：其共軛酸的 pKa 必須 > 25
常用：NaNH₂（NH₃ 的 pKa ≈ 38）、NaH、n-BuLi

10.4 炔的製備 Preparation

雙重消除反應（E2 mechanism）：

Geminal dihalide（同碳二鹵化物） → 用強鹼（NaNH₂）進行兩次 E2 → Alkyne

Vicinal dihalide（鄰碳二鹵化物） → 兩次 E2（需 anti-coplanar geometry） → Alkyne

常用過量 NaNH₂ → 若產物為 terminal alkyne 會被去質子化 → 最後加 H₂O 質子化回來

10.5 炔的還原 Reduction

這是 超級重要考點，三種還原方式要分清楚！

條件 產物 立體化學
H₂ / Pt 或 Pd（完全加氫） Alkane 烷 —
H₂ / Lindlar's catalyst 或 P-2 (Ni₂B) Cis alkene（Z） Syn addition（順式加成）
Na or Li / NH₃(l)（Dissolving metal reduction 溶解金屬還原） Trans alkene（E） Anti addition（反式加成）
Lindlar's catalyst（林德拉催化劑）：被「毒化」的 Pd 催化劑，活性降低，只能加一個 H₂ 停在 alkene。
Poisoned catalyst（毒化催化劑）：選擇性只還原炔 → 烯。
Dissolving metal reduction（溶解金屬還原）：機制經過 radical anion（自由基陰離子）中間體，因立體排斥採 trans 構型。

10.6 氫鹵化 Hydrohalogenation

無過氧化物： HX 加成 → Markovnikov 產物（X 接在取代基較多的碳上）
加兩當量 HX → geminal dihalide

有過氧化物（peroxides, ROOR）： → Anti-Markovnikov 產物（free radical mechanism 自由基機制）

10.7 水合反應 Hydration

必考 Markovnikov vs Anti-Markovnikov 的水合

條件 區域選擇性 產物（經 tautomerism）
H₂O / H₂SO₄ / HgSO₄ Markovnikov Ketone（酮）
1) R₂BH (bulky borane) 2) H₂O₂/NaOH Anti-Markovnikov Aldehyde（醛）
Keto-enol tautomerism（酮-烯醇互變異構）：enol 形式不穩定，會自動轉變為 ketone（酮式），平衡大幅偏向酮。
Enol（烯醇）：C=C 上接 -OH 的中間體。
HgSO₄：汞催化劑，補償 vinylic carbocation（乙烯基碳正離子）形成速率慢的問題。
Bulky borane（大體積硼烷）：如 9-BBN、disiamylborane，防止第二個 BH₃ 繼續反應。

10.8 鹵化 Halogenation

Alkyne + 1 eq X₂ → dihaloalkene（觀察到 syn + anti 加成混合物，機制未完全釐清）
Alkyne + 2 eq X₂ → tetrahalide（四鹵化物）
注意：alkene 的鹵化只有 anti addition（經 halonium ion），但 alkyne 的鹵化兩種都有。

10.9 臭氧分解 Ozonolysis

Ozonolysis（臭氧分解）：Alkyne + O₃ → 切斷三鍵 → 產生兩個 carboxylic acids（羧酸）

10.10 烷基化 Alkylation of Terminal Alkynes

步驟：
1) Terminal alkyne + NaNH₂ → Alkynide ion（炔化物離子，R-C≡C⁻）—— 強親核劑
2) Alkynide + methyl 或 1° alkyl halide → SN2 反應 → 新的 alkyne（碳鏈增長）

限制：2° 或 3° alkyl halide 會發生 E2 消除而非取代！
Double alkylation：用乙炔（acetylene）可以兩端各接一個烷基，但必須分步驟做（先去質子化再加烷基鹵化物，重複兩次），不能一次全加。

10.11 合成策略 Synthetic Strategies

增加飽和度： alkyne → alkene → alkane（加氫）
降低飽和度： alkane → alkene（Ch11）→ alkyne（鹵化 + 兩次脫鹵化氫）

猜題 Exam Predictions

猜題 1：還原反應的產物與立體化學

給一個 internal alkyne，分別用以下三種條件還原，畫出產物並標示立體化學（E/Z）：

  • (a) H₂ / Pd
  • (b) H₂ / Lindlar's catalyst
  • (c) Na / NH₃(l)

答題關鍵： (a) alkane、(b) cis(Z) alkene (syn addition)、(c) trans(E) alkene (anti addition)

猜題 2：水合反應的區域選擇性

給一個不對稱的 internal alkyne（如 2-butyne 或 1-propyne），分別用：

  • (a) H₂O / H₂SO₄ / HgSO₄
  • (b) 1) R₂BH 2) H₂O₂ / NaOH

畫出最終產物（含 tautomerism 後的產物）。

答題關鍵： (a) Markovnikov → ketone、(b) Anti-Markovnikov → aldehyde（只有 terminal alkyne 才能產生 aldehyde）

猜題 3：多步合成（Synthesis）

從指定的起始物合成目標分子，例如：

  • 從 1-butyne 合成 trans-2-butene
  • 從 acetylene 合成 2-hexyne
  • 從 vicinal dihalide 合成特定 alkyne 再轉化為 ketone

答題關鍵： 先建碳骨架（alkylation），再選適當試劑轉化官能基。注意 alkynide 只能跟 methyl/1° RX 反應。

猜題 4：預測產物綜合題

給一個 terminal alkyne + 過量 HBr，產物是什麼？
若加入 peroxide 條件呢？

答題關鍵： 無 peroxide → Markovnikov（geminal dihalide），有 peroxide → Anti-Markovnikov

猜題 5：酸性比較

排列以下化合物的酸性強弱：ethane、ethylene、acetylene、water、ammonia

答題關鍵： 用 ARIO，特別是 Atom（雜化軌域的 s character）：
pKa 順序：H₂O(15.7) < acetylene(25) < NH₃(38) < ethylene(44) < ethane(50)
酸性：H₂O > acetylene > NH₃ > ethylene > ethane
