Chapter 15 — IR Spectroscopy & Mass Spectrometry

Part 2：Mass Spectrometry 質譜（15.8 ~ 15.16）+ 猜題

「質譜就是把分子炸碎，看碎片的重量拼回原貌。跟我拆報廢機體逆向工程差不多。」——洛緋

15.8 質譜導論 Introduction to Mass Spectrometry

Mass Spectrometry (MS)（質譜法）：測量分子量與結構的破壞性（destructive）分析法。

原理：樣品氣化 → 用高能電子（70 eV）轟擊 → 移除一個電子 → 形成 cation radical（陽離子自由基 M⁺•） → 碎裂成碎片 → 依 m/z（質荷比） 分離偵測
Molecular ion (M⁺•)（分子離子峰）：未碎裂的分子失去一個電子後的峰，代表分子量。
Base peak（基峰）：最高的峰（定義為 100%），其他峰以此為基準計算相對強度。
Fragment ions（碎片離子）：m/z 小於 M⁺• 的峰 → 分子碎裂的產物。
M⁺• 不一定是 base peak！
例：pentane 的 M⁺• 很弱（容易碎裂），但 benzene 的 M⁺• 就是 base peak（芳香環很穩定不易碎）

15.10 (M+1)⁺• 峰 — ¹³C 同位素

¹³C 的天然豐度（natural abundance）= 1.1%

分子中碳越多 → (M+1) 峰相對於 M 峰的比例越大
估算碳數：(M+1) 峰高度 / M 峰高度 ≈ n × 1.1%（n = 碳數）
「十個碳的分子，M+1 大概是 M 的 11%。數學而已。」

15.11 (M+2)⁺• 峰 — Cl 和 Br 同位素 必考

元素同位素分布M 與 M+2 的比例
Cl ³⁵Cl = 76%、³⁷Cl = 24% M : M+2 ≈ 3 : 1
Br ⁷⁹Br = 51%、⁸¹Br = 49% M : M+2 ≈ 1 : 1（幾乎等高！）
快速判斷：
看到 M 和 M+2 兩個峰比例 ~3:1 → 含 Cl
看到 M 和 M+2 兩個峰幾乎等高 → 含 Br
沒有明顯 M+2 → 不含 Cl 或 Br

15.12 碎片分析 Analyzing Fragments

MS 只偵測帶電荷的碎片（cation），中性碎片（radical）不會被偵測到。

常見碎裂模式

碎裂類型適用官能基說明
Alpha cleavage (α-cleavage)
α-斷裂		 Alcohols、Amines、Carbonyl compounds 在官能基隔壁的 C-C 鍵斷裂
Dehydration
脫水		 Alcohols M⁺• 失去 H₂O（-18）→ 看到 M-18 峰
McLafferty rearrangement
麥克拉弗蒂重排		 Carbonyl compounds（有 γ-H 時） γ-碳上的 H 轉移到 C=O → 分子斷裂成兩個含偶數電子的碎片
Alkane 的碎裂規律：
碎片以 14 為單位遞減（CH₂ = 14）
例：hexane (M = 86) → 碎片 71, 57, 43, 29（每次少一個 CH₂ 再少一個 H）
「Alcohol 看到 M-18，那是脫水。Carbonyl 有 γ-H 就想到 McLafferty。這兩個背起來。」

15.13-15.14 高解析質譜 & GC-MS

High-Resolution MS (HRMS)（高解析質譜）：精確到小數點後 4 位
用途：區分分子量相同但分子式不同的化合物
例：C₅H₁₂ = 72.0939 vs C₄H₈O = 72.0575 → 低解析看不出差別，高解析一目了然

重要精確質量： ¹²C = 12.0000、¹H = 1.00783、¹⁶O = 15.9949
GC-MS（氣相層析-質譜聯用）：先用 GC 分離混合物（得到 retention time），再用 MS 鑑定每個成分。
應用：藥物檢測、水質、食品汙染、法醫鑑識

15.15 大分子質譜

EI (Electron Ionization)（電子游離法）：需要氣化樣品 → 大分子（蛋白質等）在氣化前就分解了，不適用。

ESI (Electrospray Ionization)（電噴灑游離法）：直接從液態噴灑游離，「softer」的技術 → 適合大分子。
MALDI (Matrix-Assisted Laser Desorption Ionization)（基質輔助雷射脫附游離法）：用雷射搭配基質 → 也適合大分子。

常搭配 TOF (Time-of-Flight)（飛行時間）質量分析器。

15.16 不飽和度 Degrees of Unsaturation (HDI) 必考計算

HDI = Hydrogen Deficiency Index（氫缺乏指數 / 不飽和度）

飽和烷烴公式：CₙH₂ₙ₊₂
每多一個雙鍵或環 → 少 2 個 H → HDI + 1
每多一個三鍵 → 少 4 個 H → HDI + 2
苯環 = 3 雙鍵 + 1 環 = HDI = 4
HDI 計算公式（含雜原子版）：

對於 CₐHᵦNᵧOᵨXₑ（X = 鹵素）：

HDI = (2a + 2 - b + c - e) / 2

規則：
• O、S → 不影響，忽略不計
• N → 加 1（因為 N 取代 C 時多帶一個 H）
• 鹵素 (F, Cl, Br, I) → 當作 H 處理（減 1）
快速記法：
O 不管、N 加一、鹵素當 H

例：C₄H₆ → HDI = (2×4 + 2 - 6) / 2 = 2 → 可能是兩個雙鍵、或一個三鍵、或一個雙鍵+一個環
例：C₆H₅Br → 把 Br 當 H → C₆H₆ → HDI = (12+2-6)/2 = 4 → benzene！

猜題 Ch15 Exam Predictions

猜題 1：IR 光譜辨識官能基

給一張 IR 光譜，要你判斷是以下哪個化合物：

  • alcohol vs carboxylic acid vs ketone vs ester

答題關鍵：

  • 有 broad O-H ~3400 + 沒有 C=O → alcohol
  • 有 very broad O-H ~2500-3300 + strong C=O ~1710 → carboxylic acid
  • 沒有 O-H + strong C=O ~1715 → ketone
  • 沒有 O-H + strong C=O ~1735 → ester

猜題 2：HDI 計算

計算以下分子式的不飽和度並推測可能結構：

  • (a) C₅H₁₀ → HDI = 1（一個雙鍵或一個環）
  • (b) C₈H₈O → HDI = (16+2-8)/2 = 5（苯環=4 + 一個 C=O）
  • (c) C₃H₅ClO → Cl 當 H → C₃H₆O → HDI = (6+2-6)/2 = 1

猜題 3：質譜同位素判讀

質譜中看到 M⁺• = 112, M+2 = 114 且兩峰幾乎等高，該分子含什麼鹵素？

答案： M : M+2 ≈ 1:1 → 含 Br。分子量 112 - 79(Br) = 33 → 可能是 CH₂=CHBr 的衍生物或類似結構。

猜題 4：碎片分析

一個 alcohol (M⁺• = 74) 的質譜中出現 m/z = 56 的強峰，這代表什麼？

答案： 74 - 56 = 18 = H₂O → 這是 dehydration（脫水）碎裂！

猜題 5：IR + MS + 分子式綜合推斷

分子式 C₄H₈O（HDI=1），IR 顯示 strong absorption at ~1715 cm⁻¹，無 broad O-H。推斷結構。

答題思路：

  • HDI = 1 → 一個雙鍵或環
  • ~1715 cm⁻¹ → C=O（ketone）
  • 無 O-H → 不是 alcohol 也不是 carboxylic acid
  • C₄H₈O + ketone → 2-butanone (MEK) 或 cyclopropane-methanol 但排除 alcohol → 答案是 2-butanone

猜題 6：波數比較題

排列以下鍵的 IR stretching 波數由高到低：C-O、C=O、C≡N、O-H、C-H

答案： O-H (~3400) > C-H (~3000) > C≡N (~2200) > C=O (~1715) > C-O (~1100)
影響因素：原子質量（越輕越高）+ 鍵的強度（三鍵>雙鍵>單鍵）
