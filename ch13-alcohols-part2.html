13.8 酚的製備 Preparation of Phenols

Phenol（酚）：工業上每年生產 200 萬噸，副產物為 acetone（丙酮）。
用途：pharmaceuticals（藥物）、polymers（聚合物）、adhesives（黏著劑）、food preservatives（食品防腐劑）

13.9 醇轉化為 Alkyl Halide

Zaitsev's rule（柴切夫規則）：E1 消除主要產生取代基最多的 alkene（most substituted = most stable）。同時偏好 trans product。

醇 → 鹵化烷（Alcohol → Alkyl halide）

醇的類型 試劑 機制
3° alcohol HX（HCl, HBr） SN1（先質子化 -OH → 離去基 H₂O → 形成 3° carbocation → X⁻ 攻擊）
1° alcohol HX SN2（先質子化 → X⁻ 背面攻擊 → inversion 構型翻轉）
1° alcohol ZnCl₂ + HCl（Lucas test 相關） SN2（ZnCl₂ 活化 -OH 為更好的離去基）
關鍵：-OH 本身是很差的 leaving group（離去基）！必須先被質子化成 -OH₂⁺ 或被轉化成更好的離去基（如 tosylate）才能離開。

13.10 醇的氧化 Oxidation of Alcohols 必考

受質 試劑 產物
1° alcohol H₂CrO₄（Chromic acid 鉻酸）/ Na₂Cr₂O₇ / Jones reagent Carboxylic acid（羧酸）（過度氧化）
1° alcohol PCC（Pyridinium chlorochromate） Aldehyde（醛）（停在這裡！）
2° alcohol H₂CrO₄ 或 PCC Ketone（酮）
3° alcohol 任何氧化劑 No reaction（沒有 α-hydrogen 可以拿掉）
PCC（Pyridinium chlorochromate 氯鉻酸吡啶鎓）：溫和的氧化劑，能把 1° alcohol 氧化成 aldehyde 而不繼續氧化成 carboxylic acid。

Chromic acid (H₂CrO₄)（鉻酸）：強氧化劑，1° alcohol 會一路氧化到 carboxylic acid。

Jones oxidation（瓊斯氧化）：用 CrO₃ / H₂SO₄ / acetone，效果等同鉻酸。
氧化程度口訣：
1° alcohol →(PCC)→ aldehyde →(H₂CrO₄)→ carboxylic acid
2° alcohol →(任何)→ ketone
3° alcohol → 不反應（沒有 α-H）

[H] 還原 / [O] 氧化 是相反方向的箭頭！

13.11 生物氧化還原 Biological Redox

NADH（菸鹼醯胺腺嘌呤二核苷酸，還原態）：生物體內的天然還原劑，遞送 hydride（H⁻）給 C=O → 產生 alcohol。

NAD⁺（氧化態）：反過來把 alcohol 氧化成 carbonyl。

NADH ⇌ NAD⁺ 的互變在代謝（metabolism）中扮演核心角色。
反應需要 enzyme（酵素）來提供選擇性（selectivity）。

13.12 酚的氧化 Oxidation of Phenol

3° alcohol 不能氧化（缺少 α-H），你可能以為 phenol 也不行——但 phenol 比 1°/2° alcohol 更容易氧化！

Phenol 氧化產物：Quinone（醌）

Ubiquinone（泛醌 / 輔酶 Q）：催化 O₂ → H₂O 的轉化，是細胞呼吸（cellular respiration）的關鍵步驟。主要存在於粒線體內膜。

13.13 合成策略 Synthetic Strategies

官能基氧化態分類：

最低氧化態 ← Alkane → Alcohol → Aldehyde/Ketone → Carboxylic acid → CO₂ → 最高氧化態

合成時先判斷起始物和目標物的氧化態差異，再選擇適當的氧化劑或還原劑。

猜題 Ch13 Exam Predictions

猜題 1：NaBH₄ vs LAH 選擇題

給一個含 ester + ketone 的分子，問：

  • (a) 用 NaBH₄ 會得到什麼產物？
  • (b) 用 LAH 會得到什麼產物？

答題關鍵： NaBH₄ 只還原 ketone → 2° alcohol（ester 不動）；LAH 兩個都還原（ketone → 2° alcohol, ester → 1° alcohol）

猜題 2：Grignard 合成設計

「用碳數 ≤ 5 的 alkyl halide 和 carbonyl compound，合成以下目標分子（一個特定的醇）」

答題關鍵：

  • 找到目標醇中 C-OH 的碳 → 這是 Grignard 攻擊 C=O 形成的鍵
  • 「斷開」那個鍵 → 一邊是 RMgX，一邊是 C=O
  • 可能有多種「斷法」→ 選碳數 ≤ 5 的組合
  • 注意：如果分子有 -OH/-NH → 要先 TMS 保護！

猜題 3：醇的氧化 — 選試劑

給出以下轉化，選擇正確試劑：

  • (a) 1° alcohol → aldehyde → 答案：PCC
  • (b) 1° alcohol → carboxylic acid → 答案：H₂CrO₄ 或 Jones
  • (c) 2° alcohol → ketone → 答案：PCC 或 H₂CrO₄ 都可以
  • (d) 3° alcohol → ? → 答案：No reaction

猜題 4：保護基的合成流程

分子同時含有 -OH 和 C=O，要用 Grignard 只攻擊 C=O。寫出完整步驟。

答案模板：

  1. TMSCl / base → 保護 -OH
  2. RMgX / ether → Grignard 攻擊 C=O
  3. H₃O⁺ workup
  4. TBAF（F⁻）→ 脫保護，還原 -OH

猜題 5：ARIO 酸性排序

排列以下化合物的酸性：ethanol、phenol、water、p-nitrophenol、acetic acid

答題關鍵：
acetic acid (pKa ~4.75) > p-nitrophenol (~7.1) > phenol (~10) > water (15.7) > ethanol (~16)
p-nitrophenol 比 phenol 更酸 → NO₂ 是強 electron-withdrawing group（Induction + Resonance 雙重穩定 phenoxide）

猜題 6：多步合成綜合題

從 alkene 出發，經過多步反應合成特定的 ketone 或 carboxylic acid。

思路模板：

  • Alkene → diol（OsO₄）或 alcohol（hydroboration-oxidation）
  • Alcohol → aldehyde（PCC）或 ketone（H₂CrO₄）
  • 或 Alkene → alkyne（halogenation + double E2）→ 再做水合得 ketone/aldehyde
